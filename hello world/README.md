# My First Project

This is a simple Hello World project written in Python.

## Files

- `hello.py` - The main Python script that prints a greeting.

## Running the Project

```bash
python hello.py
```

Or with Python 3:

```bash
python3 hello.py
```

## Output

```
Hello, World!
Welcome to my first Neovim project!
```

## About

This project was created as a first step into building projects with Neovim.
